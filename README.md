# Chatlio

Chatlio is a WhatsApp-focused Micro-CRM for sellers.

Assumptions:
- You will provide Supabase project credentials via `NEXT_PUBLIC_SUPABASE_URL` and `NEXT_PUBLIC_SUPABASE_ANON_KEY`.

Run locally:

```bash
npm install
npm run dev
```

Connect Supabase:
- Create a Supabase project and copy the URL and anon key into environment variables.

Deploy to Vercel:
- Import the repo, set environment variables, and deploy.
